La funcion de este programa es de convertir unn numero decimal a romano, en un rango de 1-1000.

1. el programa cuenta con un menu que muestra solamente 2 opciones.
2. la primera es para que el usuario ingrese un numero que este entre el 1 y 1000.
3. la segunda opcion es para terminar la ejecucion del programa
